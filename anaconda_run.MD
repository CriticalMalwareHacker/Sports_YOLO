## 7Ô∏è‚É£ Common Issues

### Camera not detected

- Ensure webcam drivers are installed
- Restart Anaconda Prompt
- Try different `--source` values

### Low FPS

- Lower resolution
- Use GPU (`torch.cuda.is_available() == True`)
- Reduce visual overlays if supported
